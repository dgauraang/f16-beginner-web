Basic Object Oriented JS

An object is a way to represent information about state and behavior in one package. Objects are fundamental to the structure of the web and most apps in general.

JavaScript objects are quite different from the objects of other languages, so let's get familiar with them. Then we can start using them to actually manipulate HTML on the page.

In Javascript, objects are essentially represented by a structure that associates keys with values. A key is a string that is the name of something and a value can be anything that you want to associate with that key. In other languages, this data structure is called dictionary or map.

As a side note, keys can be also be called fields or properties

//Create a new object with object literal
var newObj = {};

//Add a property called "five"
newObj.five = 5;
	
//Take value of "five" and add it
//to new property called "six"
newObj.six = newObj.five + 1;

//Another way to create this object
var newObj2 = {
    five: 5
};

//You can also use brackets instead of a "."
newObj["six"];  //returns 6
Try it!

Here are some exercises you can try to get familiar with Javascript objects:

  • Add a field called innerObj to newObj
  • Set this field equal to a new object with a number field and a string field
  • Access these fields with both the bracket syntax and the dot syntax

Special Objects

We've actually been using the window object the whole time! All variables declared with var live there, and any code not assigned to an object automatically gets assigned to it. Let's take a look:

var variableNamesAreHard = 1024;
window.variableNamesAreHard++; // increment the variable to 1025

//Returns 1025
window["variableNamesAreHard"];

The window object is just like that new object we made earlier, except this one has a lot of stuff predefined for you.

Try it!

Add a property to the window object and try accessing it as a local variable.

Another object of significance that's defined for you is document. This one contains all of the HTML code of the page. We can get HTML objects in our code by using a function in the object: document.getElementById. We can then access and modify the text inside using the innerText property.

Let's try it out! The next section has a lot of common HTML tags defined with ids. Pick one and try to change its contents.

The Document Object Model

The document object that you can access through the console is actually part of a larger design paradigm known as the document object model or "DOM". Initially, JavaScript and the DOM were very closely related, but both have evolved into separate entities.

The DOM is structured with a single document object and many HTML element objects. Each element also extends a more general interface known as a node. For more information, check out the MDN docs.
