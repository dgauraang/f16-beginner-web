Box Model

When the browser creates a given HTML page, it represents all the elements as boxes on the screen. Each box has four parts: content, padding, border, and margin.

Content area: space allocated for "real" content, such as text or images
Padding area: space between borders and content, used for aesthetic purposes
Border area: box including the border, which could be a physical line or invisible
Margin area: space used to separate an element and its neighbors, used for aesthetic purposes and layout

Each of the styles that change how the box looks-padding, border, and margin-can be set in CSS.

p {
    padding: 10px 5px;
    margin: 0px 10px 30px 6px;
    border: 1px solid black;
}

If four values are listed, the order is top, right, bottom, left.

If two values are listed, the first is the top and bottom and the second is left and right.

If one value is listed, they're all the same.

Display and Positioning

All HTML elements have a "display" characteristic that describes how the browser chooses to position it on the page. There are two main types of displays: block-level, inline-level.

Block elements take the whole width available and Inline elements act as plain text or take up at most the size of the content. You can also use the none property to hide an HTML element entirely.

Positioning

Float is a way to shift an element left or right relative to the current line. Because HTML positions elements vertically, you need to use tools to handle horizontal positioning.

img {
	float: right;
	margin: 10px;
}

Clear is float's complement and is used to place elements around a group of floated elements. To clear a floated group, you would clear the direction that the elements above are floated and then the element would placed as if it were in the normal HTML layout flow.

img2 {
	clear: right;
	margin: 10px;
}

Flexbox

Another common styling tool is called Flexbox. It make general page layout much easier by creating a system of rows and columns for your elements to go in.

By default, HTML arranges items top to bottom. Flexbox allows items to be arranged horizontally very easily, and even allows items to wrap to the next line.
