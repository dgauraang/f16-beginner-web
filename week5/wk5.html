Week 5

Positioning and Layout

This week, the goal is to learn some techniques to make our pages better and more powerful.

So far, all we've done is lay out the basic HTML structure and add colors and effects to the existing elements. However, it's possible to do so much more with HTML and CSS. Real webpages bend HTML elements to their will by using techniques to not only modify how HTML elements look, but also how they interact with each other.

Box Model

When the browser renders a page, it represents all of the HTML elements as boxes on the screen. Each box has four parts: content, padding, border, and margin.

Content area: space allocated for "real" content, such as text or images
Padding area: space between borders and content, used for aesthetic purposes
Border area: box including the border, which could be a physical line or invisible
Margin area: space used to separate an element and its neighbors, used for aesthetic purposes and layout

Each of the styles that change how the box looks – padding, border, and margin – can be set in CSS.

p {
	padding: 10px 5px;
	margin: 0px 10px 30px 6px;
	border: 1px solid black;
}

If four values are listed, the order is top, right, bottom, left. If two values are listed, the first is the top and bottom and the second is left and right. If one value is listed, they're all the same.

Display and Positioning

All HTML elements have a "display" property that describes how the browser chooses to position it on the page. There are two main types of displays: block-level and inline-level.

Block elements take the whole width available and inline elements take up at most the size of the content. You can also use the none property to hide an HTML element entirely.

Positioning

Float is a way to shift an element left or right relative to the current line. Because HTML positions elements vertically, you need to use tools to handle horizontal positioning.

img {
    float: right;
    margin: 10px;
}

Clear is float's complement and is used to place elements around a group of floated elements. To clear a floated group, you would clear the direction that the elements above are floated and then the element would placed as if it were in the normal HTML layout flow.

img2 {
    clear: right;
    margin: 10px;
}

Element Position

The position property is usually used to position an element outside of the flow of the page. Based on its value, the properties top, bottom, right, left change the position of the element. Its values can be:

  • relative: the position properties will offset element from where it normally lies on the page
  • absolute: the position properties specify offset from the edges of the containing block.
  • sticky: the position is relative until some special conditions are satisfied.

Check the MDN documentation for more information.

Box Sizing

The box-sizing property defines how the width and height of an HTML element are calculated.

  • content-box: width and height refer to the content box
  • border-box: width and height apply to the border box, makes it easier to size elements

Flexbox

Another common styling tool is called Flexbox. It make general page layout much easier by creating a system of rows and columns for your elements to go in.

By default, HTML arranges items top to bottom. Flexbox allows items to be arranged horizontally very easily, and even allows items to wrap to the next line.
